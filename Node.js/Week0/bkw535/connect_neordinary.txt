1. 사용자가 웹 브라우저 주소창에 https://neordinary.co.kr:443을 입력하고 Enter를 누른다.

2. 브라우저는 DNS 서버에 neordinary.co.kr의 IP 주소를 요청하고, 실제 IP 주소를 받는다.

3. 브라우저(클라이언트)는 받은 IP 주소와 443번 포트(https)를 사용해 서버와 통신을 시작한다.
   이 과정에서 TCP의 3-way Handshake 과정을 통해 연결이 설정된다.
   TCP는 sequence number를 사용해 패킷의 순서를 관리하고 데이터가 올바른 순서로 조립되도록 보장한다.
3-1. 클라이언트 → 서버 : SYN 패킷 전송 (연결 요청)
3-2. 서버 → 클라이언트 : SYN-ACK 패킷 응답 (연결 승인)
3-3. 클라이언트 → 서버 : ACK 패킷 응답 (연결 완료)

4. TCP 연결이 완료되면, 클라이언트는 HTTP GET 요청을 보낸다. 요청에는 웹사이트 주소, 브라우저 정보, 쿠키 등의 데이터가 포함된다.

5. 서버는 요청을 확인한 후, 해당하는 웹페이지 데이터를 HTTP 응답(Response)으로 반환한다. 응답에는 HTTP 상태 코드(200 OK), 웹페이지의 HTML, CSS, JavaScript 등이 포함된다.

6. 브라우저(클라이언트)는 받은 HTML, CSS, JavaScript를 해석하여 화면에 웹페이지를 표시한다.
